# Лабораторная работа 2

## Overview
Программа для работы с массивом структур NOTE . Реализует ввод 5 записей, 
сортировку по дате рождения и поиск по номеру телефона.

## Author
Валиев Владимир, 12а группа

## Usage
-  Сборка: `make`
-  Запуск: `./note_program`
-  Очистка файлов: `make clean`

## Additional Notes
- Программа разделена по модели КИС-2
- Автоматическая сборка через GitHub Actions
